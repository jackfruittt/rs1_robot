Detailed Description

Manages waypoint sequences for drone missions.

Provides sequential waypoint navigation with distance calculations and waypoint management. Supports dynamic waypoint updates and mission replanning for autonomous drone operations.

Generate random waypoints for autonomous patrol.

Parameters
start_poseStarting position for the new route
num_waypointsNumber of waypoints to generate (default: 5)
min_distanceMinimum distance between consecutive waypoints in metres (default: 1.0)
max_distanceMaximum distance between consecutive waypoints in metres (default: 3.0)
altitudeTarget altitude for waypoints in metres (default: 15.0)
Returns
Vector of randomly generated waypoints

Generates a valid patrol route with the following constraints:

  • Waypoints within map bounds (x: -50 to 50, y: -50 to 50)
  • Consecutive points separated by min_distance to max_distance
  • No path self-intersections (segments don't cross)
  • Smooth turns with angle constraints to prevent sharp reversals
